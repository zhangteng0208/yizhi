## 🛠️ 维护工具

### 数据完整性检查
```bash
./scripts/check-data-integrity.sh
```
检查所有分类的数据完整性，包括文件数量、索引一致性等。

### 自动备份
```bash
./scripts/backup-classics.sh
```
自动备份所有经典数据，保留最近30天的备份。

### 统计报告
```bash
node scripts/generate-statistics.js
```
生成详细的统计报告，包括各分类书籍数、章节数、占比等。

## 📋 数据格式

### 书籍JSON格式
```json
[
  {
    "chapter": "第1章",
    "paragraphs": [
      "段落1内容",
      "段落2内容"
    ]
  }
]
```

### 索引文件格式 (_booklist.json)
```json
[
  {
    "id": "分类前缀/书名",
    "name": "书名",
    "category": "分类名称",
    "categoryEn": "分类英文",
    "chapters": 章节数,
    "file": "书名.json"
  }
]
```
