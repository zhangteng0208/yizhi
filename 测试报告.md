## 一、项目概述

### 1.1 项目架构
- **前端**: Vue 3 + TypeScript + Vite + Vant UI
- **后端**: NestJS + TypeScript + Prisma ORM
- **数据库**: PostgreSQL
- **缓存**: Redis
- **AI服务**: DeepSeek API (通过 DMXAPI)

### 1.2 核心功能模块
项目是一个命理占卜应用，包含以下四大类功能：
- **算（suan）**: 八字、紫微斗数、奇门遁甲
- **问（wen）**: 梅花易数、六爻、易经、小六壬、周公解梦、抽签
- **相（xiang/kan）**: 面相、手相、风水
- **寻（xun）**: 寻人寻物、择日、起名、合婚、生意合伙

### 3.2 安全问题
⚠️ **严重问题**:
1. **敏感信息泄露**: `.env` 文件包含数据库密码、Redis密码、API密钥
   - DATABASE_URL 包含明文密码
   - REDIS_PASSWORD 明文存储
   - DMXAPI_API_KEY 明文存储
   - AMAP_KEY 明文存储

2. **万能验证码**: 硬编码888888，生产环境存在安全隐患

3. **JWT密钥**: JWT_SECRET 过于简单，容易被破解

⚠️ **中等问题**:
1. **CORS配置**: 未见CORS配置，可能存在跨域安全问题
2. **速率限制**: 未见API速率限制，可能被恶意刷接口
3. **输入验证**: 部分接口缺少输入验证

### 3.3 性能问题
⚠️ **潜在问题**:
1. **AI解读**: 使用SSE流式输出，但未见超时处理
2. **数据库查询**: 未见索引优化
3. **缓存策略**: Redis仅用于验证码，未充分利用

### 3.4 代码规范
✅ **优点**:
- 使用ESLint和Prettier
- 代码格式统一
- 命名规范清晰

⚠️ **问题**:
- 存在3个TODO注释未完成
- 部分文件过长（如 meihua.service.ts 305行）
